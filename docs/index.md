# FiberPath

An open, machine-agnostic filament-winding path planner.

[Documentation](fiberpath/index.md){ .md-button .md-button--primary }
[GitHub](https://github.com/fiberpath){ .md-button }

---

This is the FiberPath project hub. For now it hosts the
[software documentation](fiberpath/index.md); as the project grows, more of the
ecosystem (compatibility guidance, tooling, and related repositories) will be
gathered here.

FiberPath itself is the machine-agnostic software — it generates the winding
toolpaths. Specific machines are built by others to a documented compatibility
contract; FiberPath provides the planner and the guidance, not a single
reference machine.
